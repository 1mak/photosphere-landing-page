ELI5

What's a Merkle tree?

Imagine every photo in your library gets a unique fingerprint. Then pairs of fingerprints get combined into a new fingerprint, and so on — all the way up to one single "root" fingerprint for your whole library.

If even one photo gets corrupted, its fingerprint changes, which changes every fingerprint above it — making the problem instantly detectable. It's the same trick Git and Bitcoin use.

Get started in minutes

No accounts. No setup fee. Just your photos.

1

Clone & install

Clone the repo from GitHub and run bun install to set up in seconds.

2

Point to your photos

Connect your existing photo library. No migration, no reformatting — Photosphere works with your files as they are.

3

Browse & manage

Search, view, organize, and sync your entire photo library. Beautiful, fast, and completely yours.

terminal 
# Clone the repo
$ git clone https://github.com/ashleydavis/photosphere
$ cd photosphere

# Install & run
$ bun install
$ bun run dev
✓ Photosphere is ready.
